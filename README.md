# html_form_multiple_name

html form multiple same name submit in PHP, nodejs, golang

## Demo

### PHP

```shell
cd php
php -S localhost:8000
```

open `http://localhost:8000/`

### nodejs

```shell
cd nodejs
node index.js
```

open `http://localhost:3000/`

### golang

```shell
cd golang
go run index.go
```

open `http://localhost:8080/`
